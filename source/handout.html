Prove that if a \equiv b\pmod{n} and c\equiv d\pmod{n}, then ac\equiv bd\pmod{n}.

If a \equiv b \pmod{n}, then n | (a-b), write nk = a-b. If c \equiv d \pmod{n}, then n | (c-d), write nm = c-d. Thus, ac = (b+nk)(d+nm) = bd+bnm+dnk+n^2mk = bd+n(mb+dk+nmk). Therefore, n|ac-bd, showing that ac \equiv bd \pmod{n}.

Prove that if n>2, then there exists a prime p satisfying n<p<n!. (Hint: Consider the number n!-1.)

n!-1<n! always holds and since n>2, n! \ge 2n > n+1 so that n!-1>n. In particular, n<n!-1<n! for all n>2. If n!-1 is prime, we are done. If not, then by the FToA it has a prime factorization and each of those primes must be greater than n (or it would divide n! and n!-1, an impossibility). Those primes must also be less than n! because they divide n!-1. Hence each of those primes lie between n and n!.

Show your Euclidean Algorithm skills by using it to find \gcd(63,23). Then use the algorithm to write \gcd(63,23) as a linear combination of 63 and 23.

63 = 2\cdot 23 + 17

23 = 1\cdot 17 + 6

17 = 2\cdot 6 + 5

6 = 1\cdot 5 + 1

5 = 5\cdot 1 + 0

Thus \gcd(63,23)=1. Back‑substituting gives

1 = 11\cdot 23 - 4\cdot 63

Let n = p_1^{e_1} p_2^{e_2} \cdots p_k^{e_k} be the prime factorization of n. How many positive divisors does n have? Explain your answer.

A divisor of n must be of the form p_1^{f_1} p_2^{f_2} \cdots p_k^{f_k}, where 0 \le f_i \le e_i for each i=1,2,\dots,k. There are e_i+1 choices for each f_i. Therefore, the total number of divisors is (e_1+1)(e_2+1)\cdots(e_k+1).

Prove that if \gcd(a,b) = 1 then \gcd(a+b,ab) = 1.

(Prime solution:) If ab = 1 we are done because \gcd(a+b,ab) = \gcd(a+b,1) = 1. If ab \ne 1, let p be a prime dividing ab. By Theorem 2.27, p|a or p|b. Note that p cannot divide both a and b since \gcd(a,b)=1. In particular, p \nmid a+b. Therefore p is not a prime divisor of \gcd(a+b,ab). Since no prime dividing ab divides a+b, we have \gcd(a+b,ab) = 1.

(Non-prime way:) \gcd(a+b,a) = 1 because any common divisor of a and a+b must divide b. Similarly, \gcd(a+b,b) = 1. Therefore, by Theorem 2.28, \gcd(a+b,ab) = \gcd(a+b,a)\cdot \gcd(a+b,b) = 1.

Show that if n is composite, then n has a prime divisor less than or equal to \sqrt{n}.

Write n=ab with integers 1<a\le b. Then a^2\le n, so a\le\sqrt{n}. Any prime divisor p of a satisfies p\le a\le\sqrt{n} and therefore divides n.
